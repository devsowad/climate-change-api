## API that give climate change information

### Tech usage: nodejs, cheerio, axios, node-schedule, redis

### Run this project

In the project directory, you can run:

```shell
yarn
```

```shell
cp .env.example .env
```

```shell
yarn dev
```

Open [http://localhost:5000](http://localhost:5000) to view it in the browser.
